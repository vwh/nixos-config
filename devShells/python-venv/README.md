# Python

This is a minimal template for Python development with venv.

---

## Initialization

```bash
nix flake init -t "github:vwh/nixos-config/main?dir=nix-templates#python-venv"
```

## Usage

- `nix develop`: opens up a `bash` shell with the venv environment
