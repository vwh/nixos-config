# NodeJS

This is a minimal template for JavasScript development with NodeJS.

## Initialization

```bash
nix flake init -t "github:vwh/nixos-config/main?dir=nix-templates#nodejs"
```

## Usage

- `nix develop`: opens up a `bash` shell with the required packages
